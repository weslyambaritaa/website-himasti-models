## Syntax

### Menginstall package yang diperlukan
npm install

### Melakukan Sinkronisasi Database Master jika perlu
npm run db:sync

### Melakukan Backup Database
npm run db:backup

### Melakukan penambahan data seedmaster (provinsi dan kabupaten) cukup sekali
npm run db:seedmaster

### Melakukan penambahan data seed HakAkses cukup sekali
npm run db:seed
